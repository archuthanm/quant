1. The Objective

This project implements the foundational Modern Portfolio Theory (MPT). The primary goal was to move beyond single-asset analysis and determine how to construct an "optimal" portfolio. The analysis focuses on quantifying the relationship between expected return and risk (variance), ultimately identifying the set of portfolios known as the efficient frontier.

2. Methodology

The project was built in Python, leveraging NumPy for linear algebra and SciPy for numerical optimisation. The methodology involved several steps using a basket of S&P 500 stocks:

  • Data Sourcing: Pulled 5+ years of daily historical price data using `yfinance`.
  • Covariance Matrix: Calculated the annualised covariance matrix to capture how assets move in relation to each other.
  • Monte Carlo Simulation: Ran 50,000+ simulations with random weights to map the risk/return landscape.
  • Optimisation: Used scipy.optimise to solve for the Max Sharpe Ratio (optimal risk-adjusted) and Min Volatility portfolios.

3. Analysis: The Efficient Frontier

Plotting the results revealed the characteristic "Markowitz bullet" shape. The findings confirmed that any portfolio not on the frontier is sub-optimal. This highlights the powerful diversification benefits of combining uncorrelated assets to maximise returns for a given risk level.
